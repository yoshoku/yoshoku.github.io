Hnswlib.rb

Hnswlib.rb provides Ruby bindings for the Hnswlib that implements approximate nearest-neghbor search based on hierarchical navigable small world graphs.

Installation

Add this line to your application’s Gemfile:

ruby gem 'hnswlib'

And then execute:

$ bundle install

Or install it yourself as:

$ gem install hnswlib

Documentation

  • Hnswlib.rb API Documentation

Usage

```ruby require ‘hnswlib’

f = 40 # length of item vector that will be indexed. t = Hnswlib::HnswIndex.new(n_features: f, max_item: 1000)

1000.times do |i| v = Array.new(f) { rand } t.add_item(i, v) end

t.save(‘test.ann’)

u = Hnswlib::HnswIndex.new(n_features: f, max_item: 1000) u.load(‘test.ann’) p u.get_nns_by_item(0, 100) # will find the 100 nearest neighbors. ```
