Overview

InnerProductSpace is a class that calculates dot product for search index. This class is used internally.

Examples:

require 'hnswlib'

n_features = 3
space = Hnswlib::InnerProductSpace.new(n_features)

a = [1, 2, 3]
b = [4, 5, 6]
space.distance(a, b)
# => -31.0
